# Agent Troubleshooting Log

This document is a centralized log of issues, errors, and solutions encountered by agents during development. Before starting a complex task, review this log to avoid repeating past mistakes.

When you encounter a significant, non-obvious issue, please add a new entry.

---

## Entry Template

**Date**: YYYY-MM-DD
**Agent**: @agent-name
**Task**: [Link to task file in @agent_logs/tasks/]

### Issue

[A clear and concise description of the problem. Include error messages, unexpected behavior, etc.]

### Context

[What was the agent trying to do? What was the state of the system?]

### Solution / Workaround

[Describe the steps taken to resolve the issue. If the issue wasn't fully resolved, describe the workaround.]
